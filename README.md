# 🏥 Medical Locator

A smart Medical & Hospital Locator built using **Python, Java, SQLite, HTML, JavaScript, Leaflet.js, and OpenStreetMap**.

This application helps users find nearby hospitals, calculate real road distances, estimate travel time, and display the best driving route using live GPS.

---

## 🚀 Features

- 📍 Live GPS location tracking
- 🏥 Search hospitals worldwide
- 🔍 Smart hospital search with suggestions
- 🗺️ Interactive map powered by OpenStreetMap
- 🚗 Real road route calculation
- 📏 Accurate road distance measurement
- ⏱️ Estimated travel time
- 📌 Place a custom location anywhere on the map
- 🎯 One-click GPS centering
- 🌍 Global hospital search
- 💾 SQLite database integration
